# SRTransGAN: Image Super-Resolution using Transformer based Generative Adversarial Network

SRTransGAN is a state-of-the-art approach for image super-resolution that leverages the power of Transformers within a Generative Adversarial Network (GAN) framework. It aims to reconstruct high-quality, high-resolution (HR) images from their low-resolution (LR) counterparts with improved perceptual quality and structural consistency.

## Features

- **Transformer-based Architecture**: Both the Generator and Discriminator utilize Transformer blocks for capturing long-range dependencies and global context in images.
- **Multi-Level Generator**: The Generator ([SRTransG.py](SRTransG.py)) features a hierarchical encoder-decoder structure with skip connections for effective feature representation and reconstruction.
- **Patch-based Discriminator**: The Discriminator ([model.py](model.py)) operates on image patches, providing a more detailed and localized feedback signal for training the Generator.
- **Saliency Map Visualization**: Includes tools ([testsa.py](testsa.py)) to generate saliency maps, helping visualize which parts of the image the model focuses on during reconstruction.
- **Comprehensive Evaluation**: Supports PSNR (Peak Signal-to-Noise Ratio) and SSIM (Structural Similarity Index Measure) calculations for performance evaluation.
- **Integrated Dataset Tools**: Easy-to-use scripts for downloading and preparing the DIV2K dataset ([prepare_dataset.py](prepare_dataset.py)) and performing data augmentation ([augment.py](augment.py)).

## Working Mechanism

### Generator (SRTransG)
The Generator is designed with a multi-level Transformer architecture. It first embeds image patches and then passes them through several levels of Transformer blocks. Each level performs feature extraction at a different scale. The architecture uses:
- **Encoder**: Progressively reduces spatial resolution while increasing feature depth.
- **Decoder**: Upsamples features and combines them with encoder features through skip connections to recover spatial details.
- **Refinement**: A final set of Transformer blocks to further enhance the reconstructed HR image.

### Discriminator
The Discriminator is a Transformer-based classifier that distinguishes between real HR images and those generated by the Generator. It divides the input image into patches, projects them into a latent space, and uses a classification token with positional embeddings to produce a final validity score.

### Training Strategy
The model is trained using a composite loss function:
1. **Adversarial Loss**: Encourages the Generator to produce images that are indistinguishable from real HR images.
2. **Content Loss (L1/MSE)**: Ensures that the generated images are pixel-wise similar to the ground truth.

## Getting Started

### Prerequisites
- Python 3.8+
- PyTorch
- Torchvision
- OpenCV
- tqdm
- PIL (Pillow)
- Matplotlib
- einops

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/your-repo/SRTransGAN.git
   cd SRTransGAN
   ```
2. Install dependencies (refer to [DEPENDENCIES.md](DEPENDENCIES.md) for more details).

### Dataset Preparation
The project now supports a custom dataset structure. Ensure your dataset is organized as follows:
- `Dataset/high_resolution/`: Contains the target high-resolution images.
- `Dataset/low_resolution/`: Contains the corresponding low-resolution images.

You can also use the provided script to download and prepare the DIV2K dataset:
```bash
python prepare_dataset.py --root-dir Dataset --scales 2 4
```

### Training
To start training the model with the new dataset, run:
```bash
python main.py --epoch 200 --batch-size 4 --data-dir Dataset/high_resolution --name my_experiment
```

### Testing and Evaluation
To evaluate the trained model on your dataset:
```bash
python test.py --weight my_experiment --data-dir Dataset/high_resolution
```

### Saliency Map Generation
To generate saliency maps and visualize model attention:
```bash
python testsa.py --weight my_experiment --data-dir Dataset/high_resolution
```

================================================================================
                              CORE SCRIPTS (4)
================================================================================

1. prepare_dataset.py
   Purpose: Automatically download and prepare DIV2K dataset
   Features:
   - Downloads DIV2K training set (800 images)
   - Downloads DIV2K validation set (100 images)
   - Generates LR images at 2x and 4x scales
   - Extracts and organizes files automatically
   Usage: python prepare_dataset.py --root-dir dataset --scales 2 4

2. prepare_custom_dataset.py
   Purpose: Prepare dataset from your own images
   Features:
   - Automatic train/val/test splitting (70/15/15 by default)
   - Generates LR images for all splits
   - Supports multiple scale factors
   - Configurable split ratios
   Usage: python prepare_custom_dataset.py --source-dir /path/to/images

3. generate_lr_images.py
   Purpose: Generate low-resolution images from high-resolution images
   Features:
   - Multiple resampling methods (bicubic, bilinear, lanczos)
   - Configurable scale factors
   - Progress tracking
   - Error handling for corrupted images
   Usage: python generate_lr_images.py --hr-dir dataset/hr --lr-dir dataset/lr --scale 2

4. validate_dataset.py
   Purpose: Validate dataset structure and integrity
   Features:
   - Checks image integrity
   - Reports statistics (size, count)
   - Compares HR-LR pairs
   - Identifies corrupted files
   Usage: python validate_dataset.py --dataset-dir dataset

After running preparation scripts, you will have:

│   └── LR_x4/               # Low-resolution training images (4x)

│   ├── HR/                  # High-resolution validation images
│   ├── LR_x2/               # Low-resolution validation images (2x)

    ├── HR/                  # High-resolution test images
    ├── LR_x2/               # Low-resolution test images (2x)
    └── LR_x4/               # Low-resolution test images (4x)

================================================================================
                          TRAINING THE MODEL
================================================================================

After dataset preparation and validation:

For 2x Super-Resolution:
python main.py --data-dir dataset/DIV2K_train_HR \
               --test-dir dataset/DIV2K_valid_HR \
               --img-size 256 \
               --batch-size 4 \
               --epoch 200 \
               --name experiment_2x

For 4x Super-Resolution:
python main.py --data-dir dataset/DIV2K_train_HR \
               --test-dir dataset/DIV2K_valid_HR \
               --img-size 256 \
               --batch-size 4 \
               --epoch 200 \
               --name experiment_4x

================================================================================
                         DATASET INFORMATION
================================================================================

DIV2K Dataset:
- Training: 800 images
- Validation: 100 images
- Resolution: 2K (2040×1356 average)
- Size: ~3GB download
- Format: PNG
- Source: http://data.vision.ee.ethz.ch/cvl/DIV2K/

Recommended Custom Dataset:
- Training: 500+ images (1000+ recommended)
- Validation: 50-100 images
- Test: 50-100 images
- Min Resolution: 256×256
- Recommended: 512×512 or higher
- Format: PNG, JPG, JPEG

================================================================================
                          TROUBLESHOOTING
================================================================================

Problem: Download fails
Solution: Check internet or download manually from DIV2K website

Problem: Out of memory
Solution: Reduce batch size (--batch-size 1)

Problem: Images too large
Solution: Reduce image size (--img-size 128)

Problem: Corrupted images
Solution: Run validate_dataset.py and remove bad files
